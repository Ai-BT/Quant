# SOL SMA 골든크로스 전략

SOL 코인을 대상으로 하는 실시간 SMA (Simple Moving Average) 골든크로스 전략입니다.

## 전략 설명

- **단기 이동평균**: 5일
- **장기 이동평균**: 20일
- **매수 신호**: 골든크로스 발생 (단기선이 장기선을 상향 돌파)
- **매도 신호**: 데드크로스 발생 (단기선이 장기선을 하향 돌파) 또는 단기선 < 장기선
- **매수 금액**: 잔고의 10%씩 매수
- **체크 간격**: 5분마다 가격 확인 및 신호 체크

## 사용 방법

1. 백엔드 서버 실행
2. 프론트엔드에서 "전략 관리" 섹션 확인
3. "sol_sma_strategy" 전략 시작 버튼 클릭
4. 가상 계좌 섹션에서 거래 내역 확인

## 주의사항

- 가상 계좌를 사용하므로 실제 자금은 사용되지 않습니다
- 초기 자본은 10,000,000원입니다
- 실시간으로 Upbit API를 통해 가격을 조회합니다
