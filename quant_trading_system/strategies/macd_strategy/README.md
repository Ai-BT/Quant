# MACD + Trend Filter 전략

MACD (Moving Average Convergence Divergence)와 Trend Filter를 사용하는 실시간 트레이딩 전략입니다.

## 전략 설명

- **MACD**: 단기 EMA(12)와 장기 EMA(26)의 차이로 모멘텀 측정
- **Signal Line**: MACD의 9일 EMA
- **Trend Filter**: 200일 SMA로 큰 추세 방향 확인
- **매수 신호**: MACD 골든크로스 + 현재가 > 200일 SMA + Histogram > 0
- **매도 신호**: MACD 데드크로스 또는 현재가 < 200일 SMA
- **매수 금액**: 잔고의 10%씩 매수
- **체크 간격**: 5분마다 가격 확인 및 신호 체크

## 사용 방법

1. 백엔드 서버 실행
2. 프론트엔드에서 "전략 관리" 섹션 확인
3. "macd_strategy" 전략 시작 버튼 클릭
4. 가상 계좌 섹션에서 거래 내역 확인

## 주의사항

- 가상 계좌를 사용하므로 실제 자금은 사용되지 않습니다
- 초기 자본은 10,000,000원입니다
- 1시간봉 데이터를 사용하므로 매매 신호는 비교적 빠르게 발생할 수 있습니다
