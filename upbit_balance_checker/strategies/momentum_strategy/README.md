# 📈 모멘텀 전략 (Momentum Strategy)

일정 기간의 수익률(모멘텀)을 측정하여 매매하는 추세 추종 전략

## 📋 전략 설명

### 모멘텀이란?
**모멘텀** = N일 전 가격 대비 현재 가격의 수익률

```
모멘텀 = (현재가 - N일전 가격) / N일전 가격 × 100%

예시:
- 20일 전: 100만원
- 현재: 110만원
- 모멘텀 = (110-100)/100 = +10%
```

### 전략 로직

```
1. N일간의 모멘텀 계산
2. 모멘텀이 +X% 이상 → 매수 (상승 추세)
3. 모멘텀이 -Y% 이하 → 매도 (하락 추세)
```

## 🎯 전략 종류

### 1. 20일 모멘텀 (기본)
- **기간**: 20일
- **매수**: +5% 이상
- **매도**: -3% 이하
- **특징**: 중기 추세 포착

### 2. 10일 모멘텀 (단기)
- **기간**: 10일
- **매수**: +3% 이상
- **매도**: -2% 이하
- **특징**: 빠른 반응, 거래 빈도 높음

### 3. 30일 모멘텀 (중장기)
- **기간**: 30일
- **매수**: +8% 이상
- **매도**: -5% 이하
- **특징**: 큰 추세만 포착, 안정적

### 4. 듀얼 모멘텀
- **기간**: 20일
- **매수**: 모멘텀이 음수→양수 전환
- **매도**: 모멘텀이 양수→음수 전환
- **특징**: 추세 전환 시점 포착

## 💡 모멘텀 전략의 특징

### ✅ 장점
1. **추세 추종**: 상승 추세를 타고 수익
2. **객관적**: 명확한 수치 기반 판단
3. **단순함**: 이해하기 쉬운 로직
4. **유연함**: 기간과 임계값 조절 가능

### ⚠️ 단점
1. **횡보장 약함**: 방향성 없을 때 손실
2. **후행성**: 추세가 형성된 후 진입
3. **변동성**: 잦은 매매로 수수료 증가
4. **과거 의존**: 과거 수익률이 미래 보장 안함

## 📈 실전 활용 팁

### 1. 시장 상황별 전략

**상승장**:
```python
lookback_period = 20
buy_threshold = 0.03   # 낮게 설정 (빨리 진입)
sell_threshold = -0.05  # 높게 설정 (늦게 청산)
```

**하락장/횡보장**:
```python
lookback_period = 10    # 짧게 (빠른 대응)
buy_threshold = 0.05    # 높게 설정 (신중한 진입)
sell_threshold = -0.02  # 낮게 설정 (빠른 손절)
```

### 2. 코인별 최적화

변동성이 큰 코인:
- 기간: 짧게 (5-10일)
- 임계값: 높게 (노이즈 제거)

안정적인 코인:
- 기간: 길게 (20-30일)
- 임계값: 낮게 (작은 추세도 포착)

### 3. 다른 지표와 결합

```python
# 모멘텀 + RSI
if momentum > 0.05 and RSI < 70:
    매수

# 모멘텀 + SMA
if momentum > 0.05 and SMA20 > SMA50:
    매수
```

## 🎓 모멘텀 이해하기

### 예시 시나리오

```
날짜        가격      20일 모멘텀    신호
-----------------------------------------------
1/1        100만원     -           -
1/20       105만원    +5.0%       매수 ✅
2/10       110만원    +10.0%      유지
3/1        108만원    +8.0%       유지
3/15       95만원     -5.0%       매도 ✅
```

### 작동 원리

1. **상승 추세 감지**
   - 20일 전보다 5% 이상 상승
   - → 상승세가 계속될 것으로 예상
   - → 매수

2. **하락 추세 감지**
   - 20일 전보다 3% 이상 하락
   - → 하락세가 계속될 것으로 예상
   - → 매도
